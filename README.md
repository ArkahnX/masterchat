# MasterChat

A Node.js library for YouTube Live Chat.

## Suite

### MasterChat Core

```
npm install masterchat
```

### MasterChat CLI

```
npm install -g masterchat-cli
```

### MasterChat Desktop

macOS, Windows, Linux

> In development stage

### MasterChat Web Extension

Firefox, Chrome, Opera

> In wishlist stage
