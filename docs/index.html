Masterchat

Node.js library for YouTube Live Chat

Install

 
npm i masterchat

See MANUAL for usage.

CLI

See YouTube Live Chat through flexible filtering engine.

npm i -g masterchat-cli

See masterchat-cli for usage.

Desktop

For a desktop app, see Komet.
